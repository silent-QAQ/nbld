# 团结引擎工作流说明（已废弃）

本项目客户端主线已从 `团结引擎 + C#` 切换为 `H5 + TypeScript`。

## 当前结论

- `docs/client/tuanjie-workflow.md` 仅保留为历史说明
- 当前客户端协作约束请改看 `docs/client/web-workflow.md`
- `client/unity` 仅保留历史原型素材与脚本参考

## 废弃原因

- 当前项目阶段更适合以 H5 快速验证登录、移动、区块渲染和多人同步
- AI 驱动开发更适合纯代码、可审查、可自动化的前端工作流
- 团结引擎相关场景、资源导入和编辑器依赖不再作为当前交付路径
