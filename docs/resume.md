## 一键恢复联调

```bash

```

该脚本会：

- 启动最小服务端
- 等待当前实例健康检查通过
- 执行完整接口自检

## 云服务器协同开发

如果 H5 客户端跑在本地浏览器、Go 服务跑在云服务器，优先阅读：

```bash
docs/deployment/cloud-nat.md
```

关键原则：

- 服务监听地址用 `NBLD_GATEWAY_ADDR`
- 云服务器本机自检用 `NBLD_CHECK_BASE_URL`
- H5 客户端填外部可访问的 `Http Base Url`

## H5 客户端启动

```bash
cd client/web
npm install
NBLD_API_TARGET=http://127.0.0.1:6363 npm run dev
```

浏览器打开 Vite 输出的地址。默认请求走 Vite 代理；也可以在登录面板手动填写外部服务端地址，例如 `http://160.202.254.29:16363`。
