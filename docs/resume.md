## 一键恢复联调

```bash
bash scripts/dev_stack.sh
```

该脚本会：

- 启动最小服务端
- 等待当前实例健康检查通过
- 执行完整接口自检

## 云服务器协同开发

如果团结引擎跑在本地电脑、Go 服务跑在云服务器，优先阅读：

```bash
docs/deployment/cloud-nat.md
```

关键原则：

- 服务监听地址用 `NBLD_GATEWAY_ADDR`
- 云服务器本机自检用 `NBLD_CHECK_BASE_URL`
- 团结引擎填外部可访问的 `Http Base Url`

## 环境限制

- 当前环境无 `团结引擎` / `dotnet`
- 当前环境无 `Rust`
