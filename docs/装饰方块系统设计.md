# 装饰方块系统设计

## 概述

**本文档只关注装饰层**。游戏地图包含两层：
- **地形层**（Terrain Layer）：生物群系、主地形块（`Terrain`, `Block`字段）— **不开发破坏**
- **装饰层**（Decoration Layer）：岩石、树木、灌木等（`Decoration`字段）— **设计采集工具**

另外设计**建造系统**，让玩家通过合成和放置来构建建筑。

---

## 系统架构

```
ChunkTile 结构（当前）：
{
  x: number,
  y: number,
  terrain: string,           // 生物群系（不破坏）
  block: string,             // 地形主块如"grass"、"sand"（不破坏）
  feature: string,           // 河流等（不破坏）
  decoration: string,        // 装饰层：岩石、树、灌木（可采集）
  levelHint: string          // 难度提示
}

玩家放置的建造块（新增）：
PlacedBlock {
  id: uuid,
  mapId: string,
  x, y: int,
  blockId: string,           // 建造方块ID如"fence_wood"
  ownerID: uuid,
  timestamp: time.Time
}
```

### 工具耐久度与修复

```
赤手：无耐久限制（永久使用）
木质工具：60次使用 → 报废
石质工具：132次使用 → 报废
铁质工具：251次使用 → 报废

修复方式：
- 原始工具 + 同等原料 = 修复后工具（恢复25%耐久）
- 工作台任意组合工具（需同类工具）= 融合工具（耐久相加）
```
